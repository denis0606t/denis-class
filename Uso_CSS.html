CSS esterni e interni

È esterno un foglio di stile definito in un file separato dal documento. Si tratta di semplici documenti di testo editabili anche con il Blocco Note o TextEdit ai quali si assegna l'estensione .css.

Un foglio di stile si dice invece interno quando il suo codice è compreso in quello del documento...

Fogli collegati (esterni)

Per caricare un foglio esterno in un documento esistono due possibilità.

Usare l'elemento LINK

La prima e più compatibile è quella che fa uso dell'elemento <LINK>. La dichiarazione va sempre collocata all'interno della sezione <HEAD> del documento (X)HTML:

    <html>
      <head>
        <title>Inserire i fogli di stile in un documento</title>
        <link rel="stylesheet" type="text/css" href="stile.css">
      </head>
    <body>...
Torna all'indice

Usare @import

Un altro modo per caricare CSS esterni è usare la direttiva @import all'interno dell'elemento <STYLE>:

    <style>
      @import url(stile.css);
    </style>

Fogli incorporati

I fogli incorporati sono quelli inseriti direttamente nel documento (X)HTML tramite l'elemento <STYLE>. Anche in questo caso la dichiarazione va posta all'interno della sezione <HEAD>:

    <html>
      <head>
        <title>Inserire i fogli di stile in un documento</title>
        <style type="text/css">
          body {
            background: #FFFFCC;
          }
        </style>
      </head>
    <body>...

Fogli in linea

L'ultimo modo per formattare un elemento con un foglio di stile consiste nell'uso dell'attributo style. Esso fa parte della collezione di attributi (X)HTML definiti Common...

    <h1 style="color: red; background: black;">Esempio di stile in linea</h1>
